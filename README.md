# angularjs-web-bmi-height-weight

## Description
Calculates bmi (body mass index)
based on a person's height and weight.
Demonstrates the use of multiple models.

## Tech stack
- angular 1.6

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`  
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
